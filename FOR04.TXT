
		 Ввод информации

---------------------------------------------------------------
  Обращение		Результат
---------------------------------------------------------------

  KEY		<B> - прием байта с клавиатуры в стек. Старший
		байт стека обнуляется. Эхо режим блокирован

  GET		<B> - аналогично  KEY , но при нажатии
		клавиш CRTL и пробела отрабатывается ABORT

 DYCHECK	<B> <FFFF> - если нажата клавиша, то в стек
		записывается байт, аналогично KEY и признак
		чтения байта <FFFF>, если к моменту выполнения
		нажатий не было, то в стек записывается  <0>

  KEYD		опрос состояния клавиатуры. Если к моменты выпол-
		нения была нажата любая клавиша, то работа при-
		останавливается до следущего нажатия. При нажа-
		тии клавиш CRTL и пробела отрабатывается ABORT

 <X> <LEN>	прием строки текста в память с клавиатуры. 
  EXPECT	X - указывает начальный адрес ввода ; 
		LEN - максимальную длину вводимой строки
		( LEN < 100H ) . Ошибочно набранный символ
		стирается клавишей  DEL (ЗБ). Завершение
		ввода - клавиша  RET (ВК). Число введенных
		символов заносится в переменную SPAN

  #IN		<X> - ввод числа в стек в системе счисления,
		определяемой переменной  BASE .
		Завершение ввода - клавиша  RET  (ВК)

  Y/N		<L> - прием ответа с клавиатуры пока не будет
		нажата клавиша  Y(es) или N(o). В стеке
			L= 0 если N
			L=-1 если Y

 <N> INMEM	прямой ввод текста в память с адреса  N.
		Служебные коды заносятся в память, но не
		выодятся на монитор. Конец ввода  ^C.

 <N> KEYMEM	пердача функции клавиатуры памяти с адреса  N
		выход из режима считывания информации из памяти
		происходит при обнаружении во входном потоке
		кода  1А  или при нажатии клавиш  CRTL  и пробела

------------------------------------------------------------------

	   Вывод информации на монитор и принтер

-----------------------------------------------------------------
  Обращение	     Результат
-----------------------------------------------------------------

 <B> EMIT	вывод байта на монитор

 <B> LEMIT	вывод байта на принтер

 <B> OUT	вывод байта на устройство, определяемое перемен-
		ной LSTATE ,а именно :
		LSTATE= 0 - вывод на монитор ;
		LSTATE= 1 - вывод на принтер ;
		LSTATE=-1 - вывод на монитор и принтер.
		Зависания принтера сбрасываются нажатием клавиш
		CRTL и пробела

   SPACE	вывод пробела (кода 20H ) на монитор

 <n> SPACES	вывод n пробелов (кодов 20Н ) на устройство,
		определяемое переменной LSTATE ( см. OUT )

 " <STRING>"	вывод строки до следующего символа " на монитор

    OUT"	вывод строки до следующего символа " на устрой-
  <STRING>"	ство, определяемое переменной LSTATE 
		( см OUT )

   CR		переход к началу новой строки монитора

   LCR		переход к началу новой строки принтера

  OUTCR		переход к началу новой строки устройства,
		определяемого переменной LSTATE ( см OUT )

  <X> <LEN>	на монитор с адреса X выводится LEN
    TYPE	байтов ( LEN < 100H ). Служебные коды игнорируются

  <X> <LEN>	на принтер с адреса X выводится LEN
   LTYPE	байтов, включая служебные. Длина не ограничена

  <X> <LEN>	на устройство, определяемое переменной LSTATE
  OUTTYPE	выводится LEN байтов с адреса X, включая
		служебные. Длина текста не ограничена

  <N> DUMP	вывод содержимого паямти в  HEX  и  ASCII
		представлении с адреса  N блоками по 10Н
		байтов на устройство, определяемое переменной
		LSTATE. Продолжение вывода - клавиша пробела.
		любая другая клавиша прекращает выдачу

  LCOPY		вывод на УВВПЧ-30-004 содержиомого видеопамяти
		в пределах с  0-ой по  20-ую строки. При работе 
		в режиме высокого разрешения копия получается
		"лежа" на листе

  GRAF0		установка  0-ой страницы монохромного режима
		среднего разрешения

  GRAF1		установка  1-ой страницы монохромного режима
		среднего разрешения

  GRAFHR	установка монохромного режима высокого разрешения

  GRAFC
	установка цветного режима среднего разрешения

   CLS		очистка окна монитора

  CLSTR		очистка окна монитора от текущей позиции маркера
		до конца окна

 <X> <Y> POZH	установка маркера в позицию X (по гор.)
		и Y (по верт.). При выходе аргументов за
		пределы окна,  маркер устанавливается в
		крайнюю допустимую позицию

    POS		<X> <Y> - в стек считываются горизонтальная
		 и вертикальная позиции маркера

 <X> TABX	горизонтальная табуляция маркера в позицию X

  UP/DOWN	переключение регистров с верхнего на нижний
		и обратно

  INVERS	переключение маски вывода на инверсную и обратно

  BLOCKM	запрещение индикации маркера

  DEBLOC	разрешение индикации маркера

  DUPLEX	Эхо-режим. Код каждой нажатой клавиши выдается
		на устройство, определяемое переменной LSTATE.
		Выход из Эхо-режима при нажатии ^C

 <X> GROUND	установка цвета фона и выбор палитры. Байт с
		вершины стека заносится в регистр управления 
		цветом

	Вывод 16-ти разрядных чисел в соответствии
		с выбранной системой счисления

  <X> .		вывод числа из стека со знаком

  <X> U.	вывод числа из стека без знака

  <X> ?		вывод числа из памяти по адресу X со знаком

  <X> U?	вывод числа из памяти по адресу X без знака

 <X> PRINT	форматированный вывод числа из стека без подав-
		ления старших нулей на устройство, определяемое 
		переменной LSTATE (см OUT ). Число выводимых
		младших разрядов определяется переменной  FORMAT

   Обращение	   Результат

		вершину стека операндов
