              
	 Пересылки, чтение и запись в память

-------------------------------------------------------------
 Обращение	      Результат
-------------------------------------------------------------


  <X> @ 	<@X> - на вершине стека адрес заменяется 16-ти
		разрядным числом, сохраняющимся в памяти

  <X> C@	<C@X> - на вершине стека адрес заменяется байтом 
		из памяти ; старший байт вершины обнуляется

 <Y> <X> !	16-ти разрядное число Y записывается по
		адресу X ; вершина и подвершина стека сбрасываются

 <Y> <X> C!	младший байт Y записывается по адресу X ;
		вершина и подвершина стека отбрасываются

 <X> <Y> <N>	пересылка N байтов с адреса X на адрес
   CMOVE	Y. Пересылка идет с первого элемента

 <X> <Y> <N>	пересылка аналогично CMOVE с последнего
   CMOVE>	элемента

 <B> <X> <N>	заполнение N байтов памяти с адреса X
   FILL		кодом B


 <X> DUP	<X> <X> - дублируется вершина стека

 <Y> <X> DROP	<Y> - сбрасывается вершина стека

 <Y> <X> SWAP	<X> <Y> - обмениваются вершина и подвершина стека

 <Y> <X> OVER	<Y> <X> <Y> - дублируется подвершина стека

 <Y> <X> PRESS	<X> - удаляется подвершина стека ; анало-
		гично исполнению пары SWAP DROP

 <Y> <X> 2DUP	<Y> <X> <Y> <X> - дублируются вершина и подвершина
		стека ; аналогично исполнению пары OVER OVER

 <Z> <Y> <X>	<Y> <X> <Z> - циклическая перестановка трех
   ROT		элементов стека

 <Xn>...<X0>	<Xn+1> <Xn-1>...<X0> <Xn> -  циклическая пере-
   <n> ROLL	 становка элементов стека, при которой на вер-
		шину помещается n-ый элемент, удаляемый из
		стека . Для вершины стека принято n=0 , для
		подвершины n=1 . Таким образом :
		0 ROLL - не производит никаких действий ;
		1 ROLL - аналогично SWAP ;
		2 ROLL - аналогично  ROT

 <Xn>...<X0>	<Xn>...<X0> <Xn> - на вершине стека дублируется
   <n> PICK	 n-ый элемент стека. Таким образом :
		0 PICK - аналогично  DUP
		1 PICK - аналогично  OVER

  S0		<S0> - в стек помещается адрес дна стека
		операндов

  S@		<S> - в стек помещается адрес текущего положе-
		ния указателя стека операндов

  <X> S!	адрес текущего положения стека операндов
		заменяется на число  X , взятое со стека

  STEK		на монитор выводится сообщение о глубине за-
		полнения стека операндов ( используется при 
		отладке программ )

  R@		<R> - дублируется в стек операндов адрес с
		вершины стека возвратов

  R>		<R> - адрес с вершины стека возвратов удаляется
		и переносится на вершину стека операндов

  >R		адрес из стека операндов переносится на вершину
		стека возвратов
